Portfolio Website Setup Instructions

1. Required Files:
   - index.html (main website file)
   - config.php (database configuration)
   - process_contact.php (contact form handler)
   - images/ (folder containing all images)

2. Database Setup:
   - Database name: portfolio
   - Username: root
   - Password: 182315
   - Run create_database.sql in phpMyAdmin to set up tables

3. How to Run:
   - Install XAMPP
   - Copy all files to C:\xampp\htdocs\potof
   - Start Apache and MySQL in XAMPP
   - Access website at http://localhost/potof/index.html

4. Important Notes:
   - Make sure all image paths are correct
   - Keep the database credentials secure
   - Test the contact form after setup
